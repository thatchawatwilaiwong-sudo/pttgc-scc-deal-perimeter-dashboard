JV Logic

    Key DD Question

    • ROC จะเข้า JV ได้ก็ต่อเมื่อมี feedstock plan และ offtake / balancing plan ที่ชัดเจน
    • Narrow case ลด downstream complexity แต่ยังไม่ตอบ customer / polymer integration เต็มที่
    • Base case เหมาะสุดสำหรับ industrial logic เพราะรวม olefins + PE/PP outlet โดยตรง
    • Extension case ต้องระวัง valuation, governance, minority consent และ asset ที่ไม่ใช่ direct olefins outlet
